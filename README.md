# 🚗 Car Dodge Console Game 🏁

Dodge incoming cars in this simple C++ console game!  
Move your car left and right to avoid collisions and rack up the score.  

---

## 🎮 Controls

| Key       | Action        |
|-----------|---------------|
| A / a     | Move Left ⬅️   |
| D / d     | Move Right ➡️  |
| Esc       | Quit Game ❌   |

---

## ⚙️ Requirements

- Windows OS 🪟  
- C++ compiler (MSVC or MinGW)  
- `<conio.h>` & `<windows.h>`

---

## ▶️ Build & Run

```bash
g++ car_game.cpp -o car_game.exe
car_game.exe
